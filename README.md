# Sobre o projeto

- O aplicativo consiste em exibir um Dashboard de Vendas.

# Tecnologias

Para a construção deste projeto, foram utilizadas as seguintes ferramentas:

Back-end:
- Java
- Spring Boot
- H2
- PostgreSQL
- Heroku CLI

Front-end:
- TypeScript
- ReactJS
- Axios
- React Router DOM
- Bootstrap
- ApexCharts
- Netlify
